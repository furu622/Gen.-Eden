# Gen. Eden 🍃
知識ゲームアプリ — *Gen. Eden*

© 2026 ふる. All Rights Reserved.

---

## 🎮 概要

**Gen. Eden** は、科学史・芸術史・一般歴史をテーマにした  
ブラウザ完結型の知識ゲームです。

英語の問題文を読みながら、
制限時間内に選択式で回答していきます。

学習というよりも、
「気軽に知識を回す」感覚で遊べる設計になっています。

---

## ✨ コンセプト

- 重すぎない
- 作業感を出さない
- 短時間でも遊べる
- 英語に自然に触れられる

知識を“鍛える”というより、
知識に“触れ続ける”ための小さな環境です。

---

## 🚀 使い方

1. `index.html` をブラウザで開く
2. カテゴリを選択
3. 難易度を選択
4. 制限時間内に答える

インストール不要・ログイン不要。
すぐ遊べます。

---

## 📚 現在のカテゴリ

### 科学史 – 力学（Mechanics）


今後カテゴリは拡張予定です。

---

## ⚙️ 主な機能

- 選択式クイズ
- 難易度別タイム制限
- セッション統計表示（正解数 / 経過時間）
- 英語音声読み上げ
- 解説表示
- 完全クライアントサイド動作

---

## 📊 セッション表示

画面右上に以下を表示：

- 正解数 / 総問題数
- 経過時間

1セッションごとに結果が表示されます。

---

## 🛠 技術構成

- HTML
- CSS
- Vanilla JavaScript
- Web Speech API（音声読み上げ）

フレームワーク・ビルド環境なし。
シンプルな構造です。
